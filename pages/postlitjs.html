postLit.js is a simple NPM module for interacting with the postLit API.

postLit.js is a simple NPM module for interacting with the postLit API. It is constantly evolving and is very easy to use. The fact that it is official helps it because it can be maintained at pace with the site itself, without being very far behind on many features.

postLit.js is used by many bots on the postLit website, all of which are allowed as long as they follow the bot guidelines.
